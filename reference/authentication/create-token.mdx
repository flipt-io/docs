---
title: "Create Token"
openapi: POST /auth/v1/method/token
---

<RequestExample>

```bash cURL
curl --request POST \
  --url https://try.flipt.io/auth/v1/method/token \
  --header 'Content-Type: application/json' \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer <token>' \
  --data '{
  "description": "<description>",
  "name": "<name>"
}'
```

</RequestExample>

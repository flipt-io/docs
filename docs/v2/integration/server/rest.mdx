## Overview

For server-side applications, Flipt v2 provides a REST API for evaluating flags. The REST API SDKs are available in the following languages:

<Tip>
  These SDKs are the same ones that are available for the [Flipt v1 REST
  API](/v2/integration/server/rest) and are backward and forward compatible
  which means there is no need to rewrite your existing code to integrate with
  Flipt v2.
</Tip>
